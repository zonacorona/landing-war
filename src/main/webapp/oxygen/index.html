To Install Oxygen and the Rackspace/OpenStack Add-ons

For more information about the Rackspace/OpenStack tool-chain see the Writer's Guide.

  1. Download the Rackspace/OpenStack Global Options File to a convenient location on your computer.
  2. If you have not already done so, Install Oxygen Editor or Oxygen Author from the Oxygen Web site. Be sure to download the version for which you have a license and download version 14 or higher.

  3. Launch Oxygen.

    1. When prompted, paste in your license.

    2. Note: This step overwrites any global settings you have already configured in your local oXygen installation. If you prefer not to overwrite your settings but still want to add the Rackspace/OpenStack frameworks, in oXygen click Options -> Preferences, Add-Ons, then click Add, paste http://docs.rackspace.com/oxygen/updates/updates.xml into the field, and click OK. Proceed to Step 4.

      After Oxygen opens, click Options -> Import Global Options....

    3. Navigate to the global-options.xml file you downloaded in Step 1 above and click Open.

    4. Load the Rackspace/OpenStack add-ons:

      1. Click Help -> Check for Add-ons updates....
      2. Click Review Updates.
      3. Click the check boxes for wadl-tools and rackbook.
      4. Click Install.
      5. Click the box to accept the license and click Continue.
      6. Click OK and then restart Oxygen to load the new frameworks.
